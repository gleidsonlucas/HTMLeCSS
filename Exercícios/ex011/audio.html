Áudio em html

Para usarmos áudio, precimos abrir a tag <audio>.
Indico usar o preload= "none". Assim, o áudio só carregara se o usuário apertar play. Use também o parâmetro controls.

Acessibilidade

Dentro da tag <audio> podemos adcionar vários <source> caso o .mp3 não funione.

Parâmetro LOOP

Casoo você use o parâmetro "loop", assim que o audio acabar, já começará sem o usuário dar play!
